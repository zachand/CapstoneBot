this only works on windows at the moment.
1. make sure you have both NodeJS and NPM (Node Package Manager) installed
2. follow this guide to get your bot account set up
    https://discordpy.readthedocs.io/en/stable/discord.html
3. when you have your token from the above step put it in a file called ".env"
    the only line in this file should be:
    TOKEN="token goes here keep the quotes"
4. run "winrun.bat"
